material-progress

Usage

    <material-progress value="25" max="100" circle="64"></material-progress>

Attributes

  • value: Number, current value of the progress indicator
  • max: Number, maximum value of the progress indicator
  • circle: Number, will make the progress indicator appear as a circle, the value is the radius in pixels

Styling

  • width of progress bar: --progress-bar-width, default: 100%
  • height of the progress bar: --progress-bar-height, default: 5px
  • color of the progress indicator: --progress-bar-color, default: rgb(0, 188, 212)
  • background color of the progress indicator: --progress-background, default: #eeeeee
  • font size of the progress bar when circular: --progress-font-size, default: 4px
